# C# ile Sıfırdan Mini Deep Learning Framework

Bu proje, modern deep learning framework’lerinin (PyTorch / TensorFlow gibi) arka planda nasıl çalıştığını anlamak amacıyla **tamamen sıfırdan C# kullanılarak geliştirilmiş eğitim amaçlı bir neural network implementasyonudur**.

Hiçbir makine öğrenmesi kütüphanesi kullanılmamıştır.
Amaç bir framework üretmek değil, **deep learning’in temel matematiğini ve mühendisliğini doğrudan kodlayarak öğrenmektir.**

İlham Kaynağı
Bu çalışma özellikle aşağıdaki eğitim yaklaşımından ilham almıştır: 

Andrej Karpathy — neural network eğitim içerikleri ve micrograd yaklaşımı

Amaç konsepti anlamak ve aynı mantığı farklı bir dilde yeniden inşa etmektir.

Deep learning eğitim projelerinin büyük çoğunluğu Python ile yapılmaktadır.

Bu proje şunu göstermeyi amaçlar:

Deep learning bir dil özelliği değil, matematik ve mühendislik problemidir.

Aynı prensipler C#, Java veya başka herhangi bir dilde uygulanabilir.
